# Pila

Pila is a simple Forth interpreter written in Odin. It was created as an exercise to learn the Odin programming language and explore interpreter design.

## Features

- Forth-inspired stack-based execution model
- Basic word definitions and control flow
- Interpreter implemented using Odin

## Usage

Build and run the interpreter with the Odin compiler:

```sh
odin build .
./pila
```

Or run a specific Pila source file if supported by the interpreter:

```sh
./pila test.pila
```
